# BCC · 資訊科技課程作業系統

學生作業上傳 + 老師管理系統。前端純 HTML/JSX,後端用 Google Apps Script。

## 給學生的網址

部署完成後,學生用這個網址:

```
https://你的帳號.github.io/bcc/
```

## 給學生的初始登入資訊

- 學號:他自己的學號
- 預設密碼:`00000000`
- 第一次登入會被強制改密碼,並設定 Email

## 老師後台

開啟系統 → 右上「老師後台」→ 輸入 TEACHER_KEY

TEACHER_KEY 在 Apps Script 的「專案設定 → 指令碼屬性」可以查看 / 修改。

## 後端

後端程式不在這個 repo,請勿上傳。後端在 Google Apps Script,部署 URL 在使用者第一次開啟系統時會被要求輸入並儲存到瀏覽器。
